# Delete Production Submissions Plugin for OMP 3.4

This plugin allows Press Managers and Administrators to delete submissions that have reached the **Production** stage in **Open Monograph Press (OMP) 3.4**.

## Features

-   **Targeted Deletion**: Specifically designed to handle submissions in the Production stage which can sometimes be difficult to remove via the standard interface.
-   **Safe Management**: Accessible only to authorized users via the Plugin management interface.
-   **Modal Interface**: Provides a clean AJAX-based modal for selecting and deleting submissions.
-   **OMP 3.4 Optimized**: Built to work with the latest PKP library standards in OMP 3.4.

## Installation

1.  Download the `deleteProductionSubmissions.tar.gz` package.
2.  Log in as a Site Administrator.
3.  Go to **Settings > Website > Plugins**.
4.  Click on **Upload A New Plugin**.
5.  Select the `.tar.gz` file and click **Save**.
6.  Alternatively, extract the package and upload the `deleteProductionSubmissions` folder to your `plugins/generic/` directory.

## How to Use

1.  Navigate to **Settings > Website > Plugins**.
2.  Locate the **Delete Production Submissions** plugin and ensure it is enabled.
3.  Click the blue arrow next to the plugin name and select **Deletion**.
4.  Follow the prompts in the modal to identify and remove production submissions.

## Requirements

-   **OMP 3.4.x**
-   PHP 8.0 or higher.
